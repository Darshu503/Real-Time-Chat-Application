# Real-Time Chat Application

This is a real-time chat application built using the MERN stack.

## Features
- User authentication (Login/Signup)
- Real-time messaging
- Media sharing
- User profile management
- Change themes

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time: Socket.io

## Learning Note
This project was built while learning the MERN stack and implementing real-time features.
